GCP翻訳APIサンプル

概要

このプロジェクトは、Google Cloud Platform（GCP）の翻訳APIを使用して、日本語のテキストを英語に翻訳するサンプルアプリケーションです。

機能

  • 日本語テキストファイルを英語に翻訳
  • HTML形式の保持
  • 翻訳結果の自動保存

必要条件

  • Python 3.6以上
  • GCPアカウントとプロジェクト
  • 有効なAPIキー

セットアップ

  1. このリポジトリをクローンする

    git clone <repository-url>

  2. 必要なパッケージをインストール

    pip install -r requirements.txt

  3. .envファイルを作成し、以下の内容を追加

    GOOGLE_CLOUD_PROJECT_API_KEY=あなたのGCP_APIキー

使用方法

  1. templates/ja.txtファイルに翻訳したい日本語テキストを記入します。

  2. スクリプトを実行します。

    python ja_to_en.py

  3. 翻訳結果はresults/en.txtに保存されます。

注意事項

  • APIキーは公開しないでください
  • GCPの無料枠を超えると料金が発生します
  • 大量のテキストを翻訳する場合は、GCPの利用制限を確認してください
